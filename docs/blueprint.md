# **App Name**: Career Compass

## Core Features:

- Skills and Interest Analyzer: Gathers user inputs and analyzes them to determine the best career matches.
- Personalized Career Recommendations: Provides 3-5 tailored career paths based on user's profile.
- AI-Powered Skill Gap Analysis: Uses generative AI tool to identify and explain the skill gaps for each career.
- Roadmap Generator: Outlines a step-by-step learning and career plan with estimated timelines.
- Trend Adaptation: Adapt career suggestions and technologies based on current industry trends.
- Resource Links: Show curated resources, learning materials, certifications, and job boards.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of trust and expertise. Note: currently only Google Fonts are supported.
- Background color: Light gray (#E0E0E0) for a clean and neutral backdrop.
- Accent color: Dark blue (#000080) for highlighting key elements and CTAs.
- Headline font: 'Space Grotesk', a sans-serif font that provides a computerized, techy feel, while keeping readability at its finest. Note: currently only Google Fonts are supported.
- Body font: 'Inter', also sans-serif, matches Space Grotesk aesthetically while also being extremely readable, creating harmony in typography. Note: currently only Google Fonts are supported.
- Use simple, professional icons related to different skills, careers, and industries.
- A clean and intuitive layout with a focus on readability and user experience. Consistent use of spacing and alignment to create a visually appealing interface.
- Subtle animations to provide feedback and guide users through the career planning process.
